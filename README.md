# ✈️ Airport Data Analysis System (Python)

## 📌 Overview

This project is a Python-based system that analyzes airport flight departure data from CSV files. It performs statistical analysis and visualizes results using histograms.

## 🚀 Features

* User input validation (airport code & year)
* Data analysis (flights, delays, distances, etc.)
* Saves results to a text file
* Histogram visualization using graphics.py
* Supports multiple dataset processing (loop system)

## 🛠️ Technologies Used

* Python
* CSV file handling
* graphics.py (for visualization)

## 📊 Example Outputs

* Total flights
* Percentage of delays
* Airline-specific analysis
* Weather-based insights

## ▶️ How to Run

1. Place CSV files in the same folder
2. Run the Python file:

   ```
   python w2119784.py
   ```
3. Follow on-screen instructions

## 📁 Project Type

This is mainly a **back-end data processing system** with a simple front-end interface.
